CSS reset

[TOC]

前言：

CSS作为控制HTML/XHTML的样式和布局的语言，在当今网页前端开发中至关重要。CSS目前最新版本为CSS5，更是能够真正做到网页表现与内容分离。但当今几款主流的浏览器（Chrome、FireFox、Opera、Safari、IE）都有一套自己理解CSS规范的方法，对默认样式有不同的默认设置值。这会导致同一页面在不同的浏览器下，有不同的效果。这就是浏览器兼容性问题之一，而CSS reset就是可以解决这一问题的方法。

CSS reset简介：

CSS reset也叫CSS重设、CSS复位。由于不同浏览器解释CSS样式的默认设置值不同，导致开发人员在没有定义某个CSS属性时，不同的浏览器会按照自己的默认设置值来为没有定义的样式赋值，所以我们在页面开发前预先定义好一些CSS样式，来让所有浏览器都按照同样的规则解释CSS，这样就能避免发生这种问题。
